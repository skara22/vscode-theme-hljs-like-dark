# HLJS-like Dark
### Dual-language description (EN / SK)

A clean dark Visual Studio Code theme inspired by **highlight.js**, fine-tuned for **C++**, **HTML**, **CSS**, **PHP**, **JavaScript**, **JSON/JSONC**, and **TypeScript** syntax highlighting.

The goal of this theme is to deliver a **balanced**, **readable**, and **consistent** look between semantic and TextMate tokens ‚Äî with precise control over punctuation, parameters, and keywords.

---

## üé® Features

- Consistent **C++ syntax highlighting**
- Accurate colors for **types, keywords, literals, and templates**
- Hand-tuned punctuation and operator colors
- Unified semantic + token-based color logic
- Clean dark UI background matching code contrast
- Smooth integration with VS Code dark UI

---

## üß© Installation

1. Download or install the theme from the [VS Code Marketplace](https://marketplace.visualstudio.com/).
2. Open the **Command Palette** (`Ctrl+Shift+P`).
3. Run **Preferences: Color Theme**.
4. Choose **HLJS-like Dark** from the list.
